# Game Sign System

[English](README_EN.md) | [中文](README.md)

> ⚠️ **Disclaimer**: This project is partially constructed using **Gemini 3.0 Pro** and **GPT-5.1 Thinking**.

This is an open-source tournament sign-in and management system, featuring a Web Admin Panel and an Android Client App.

## Structure

*   `web_server/`: The Python Flask backend and web interface.
*   `android_app/`: The Android client application (Jetpack Compose).

## Features

*   **Player Sign-in**: QR code scanning or manual input.
*   **Tournament Management**: Admin panel to manage players, groupings, and matches.
*   **Real-time Updates**: Web sockets or polling for live status updates.
*   **Android App**: Native Android app for players to check status and sign in.
*   **Responsive Web UI**: Material Design 3 style web interface.
